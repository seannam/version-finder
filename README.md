# Version Finder
How to find the version of programming languages and dev tools

## Programming Languages
Python: `python --version` or `python -V`  
Ruby: `ruby --version` or `ruby -v`  
Java: `java -version`  

## Frameworks
Node: `node --version` or `node -v`  

## Libraries 
React:  `react -v`  
React Native:  `react-native -v`
